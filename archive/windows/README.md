# Windows Version Archive

This directory contains the original Windows version of Jellyfin Music Organizer.

## Contents

- `Jellyfin Music Organizer.exe` - Windows executable (v3.06)
- `Build Files/` - Original source code used to build the Windows version

## Note

This project has been migrated to Linux (Ubuntu 24.04). The Windows version is archived here for reference and legacy support.

If you need to use the Windows version:
1. Download `Jellyfin Music Organizer.exe`
2. Run it directly on Windows (no installation required)

For the current Linux version, please see the main README in the repository root.
